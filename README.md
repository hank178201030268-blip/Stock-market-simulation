# Stock Market Simulation

This is a simple stock and crypto simulation website with:

- TW stocks
- US stocks
- Crypto assets
- simulated buy/sell trading
- portfolio and PnL tracking

## Run

1. Install dependencies

```bash
npm install
```

2. Start app

```bash
npm start
```

3. Open browser

```text
http://localhost:3000
```

## Notes

- The app tries to fetch live market prices from public data sources.
- If an API source fails or rate limits, it falls back to seeded simulated prices.
- This is a learning/demo app, not financial advice or real trading software.
